# Prueba-de-desempe-o-M1
 Students managment

This program is designed to register and manage Students
It uses json data to store information, and organizes the code into modules.
Navigation is handled through a menu, where you can select the desired action.
Features

    1.Register Students 
    - Register Students using their ID, first name, age, cours and confirm if this student stay active or inactive complying with these conditions program  confirms the entry.

    2.List Students 
    - Reed students_json in json archive and show the students in the console if no student is found, it means there are no students registered yet.

    3.Search Students 
    - Search a students whith option 1 id or option 2 name once a valid option has been chosen shows all students saved in data.jason.

    4.Update Students 
    - in this option the program request the student ID to be updated then it asks you for all the information so you can update the one you want after entering all the updated data, it informs you that the process was successful.

    5.Delete Students 
    - The delete option asks for the student ID to be deleted and then shows you a yes or no confirmation to make sure that it is the ID of the student you are going to delete if the confirmation is positive, the student is eliminated; if it is negative, the process is stopped.

    6.exit
    -Option exit close the program and show the thank you massage
